# meelia

## 概要
簡易オレオレフレームワーク。  
CodeIgniter2.xとかSymfony1.xをパク・・・参考に作った。  
GithubとTravisとGitの練習も兼ねてる。


## 初期設定
    git clone https://github.com/Alfr0475/meelia.git
    cd meelia
    curl -s http://getcomposer.org/installer | php
    php composer.phar install
